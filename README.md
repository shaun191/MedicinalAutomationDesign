# Medicinal Automation & Design

Production-ready Next.js site for Medicinal Automation & Design.

## Run locally

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Deploy to Vercel

1. Push this folder to GitHub.
2. Import the repository in Vercel.
3. Set the production domain to `www.medicinalautomationdesign.com`.
4. Deploy.

## Notes

- SEO metadata is in `app/layout.js`.
- Schema JSON-LD is in `app/page.js`.
- Main site content is in `components/Site.jsx`.
- Logo and favicon are in `/public`.
- The current form uses `mailto:` for launch safety. Replace with Microsoft Forms, Dynamics, HubSpot, or a custom form endpoint when ready.
